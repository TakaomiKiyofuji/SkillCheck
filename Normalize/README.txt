【概要】
このプログラムは、TSV（タブ区切り）形式のデータに対して「第一正規化（1NF）」を適用します。
1つのセルにコロン（:）で区切られた複数の値が含まれている場合、それぞれの値を分割し、
行を増やすことで1セル1値の構造に変換します。

【実行方法】
run_Normalize.bat をダブルクリックすることで、以下の処理が自動で実行されます：
- NormalizeTSV.java のコンパイル
- input.tsv の読み込み
- output.tsv への変換結果の出力

【使用ファイル】
- NormalizeTSV.java：第一正規化処理のソースコード
- input.tsv：正規化前の入力データ（例）
- run_Normalize.bat：コンパイルおよび実行を自動化するバッチファイル

【実行環境】
- Windows 10/11
- Java 17 以上（JDKがインストールされていること）
- コマンドプロンプトまたは PowerShell

【注意事項】
- 入力ファイルは UTF-8 で保存してください。
- 入出力ファイル名は固定（input.tsv / output.tsv）ですが、ソースを編集すれば変更可能です。

【Windows で実行する場合の重要な注意点】
-出力が文字化けすることを防ぐため、実行前にコマンドプロンプトで以下を入力してください

chcp 65001


これにより、コンソールのコードページが UTF-8 に切り替わり、日本語や記号が正しく表示されるようになります。
文字化けが発生した場合は、ターミナルの種類やフォント設定も併せてご確認ください。
