【概要】
このプログラムは、TSV（タブ区切り）形式で「キー」と「値」の2列からなるデータを読み取り、
同一キーを持つ値をコロン（:）で連結して、1行にまとめることで第一正規化の逆変換を行います。

【実行方法】
run_Denormalize.batをダブルクリックすることで、以下の処理が自動で実行されます：
- DenormalizeTSV.java のコンパイル
- input.tsv の読み込み
- output.tsv への変換結果の出力

【使用ファイル】
- DenormalizeTSV.java：逆正規化処理のソースコード
- input.tsv：キー・値の2列の入力データ（例）
- run_Denormalize.bat：コンパイルおよび実行を自動化するバッチファイル

【実行環境】
- Windows 10/11
- Java 17 以上（JDKがインストールされていること）
- コマンドプロンプトまたは PowerShell

【注意事項】
- 入力ファイルは UTF-8 で保存してください。
- 入出力ファイル名は固定（input.tsv / output.tsv）ですが、ソースを編集すれば変更可能です。

【Windows で実行する場合の重要な注意点】
-出力が文字化けすることを防ぐため、実行前にコマンドプロンプトで以下を入力してください

chcp 65001


これにより、コンソールのコードページが UTF-8 に切り替わり、日本語や記号が正しく表示されるようになります。
文字化けが発生した場合は、ターミナルの種類やフォント設定も併せてご確認ください。
